== Description ==



The Facebook Like Widget plugin adds a Facebook profile widget to your WordPress site along with the option to include the Facebook Like Box.
== Installation ==



1. Upload the Addon to the "/wp-content/plugins/" wordpress directory.

2. Activate the plugin through the 'Plugins' menu in WordPress
